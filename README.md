Файл посвящён классификации изображений и экспериментам с архитектурами CNN, включая базовые модели, трансфер-лернинг и продвинутые варианты.
В качестве данных был взят датасет с новостями с ресурса Lenta.ru, размеченный по категориям новостей.
https://www.kaggle.com/datasets/yutkin/corpus-of-russian-news-articles-from-lenta
Содержание:
Подготовка данных
- Кастомный Dataset для загрузки картинок по папкам классов.
- Аугментации с albumentations: кропы, отражения, повороты, яркость/контраст, CoarseDropout и нормализация.
Модели классификации
- BasicCNN — простая свёрточная сеть без усложнений.
- TransferResNet — ResNet18 с предобученными весами (ImageNet), используется как фиксированный feature extractor.
- AdvModel — кастомная глубокая CNN с residual-блоками, батчнормом и дропаутом.
Обучение
- Реализована функция fit_one_cycle (OneCycleLR + warmup).
Метрики: accuracy.
Дополнительно
Реализация GAN (DCGAN) пока не заработал нормально:
- Generator для генерации изображений.
- Discriminator для различения реальных и синтетических картинок.
